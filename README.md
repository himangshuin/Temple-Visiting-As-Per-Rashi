# 🕉 Temple Visiting as Remedy

A beautifully designed web app that helps users discover the most suitable temples to visit based on their **Vedic Birth Sign (Rashi)**.

The app includes sacred temple recommendations from multiple Hindu traditions, including:

* Shiva Jyotirlingas
* Devi Shaktipeeths
* Ganesha Temples
* Vishnu Temples
* Rama & Hanuman Temples
* Krishna & Radha Temples

Based on the spiritual research and teachings of **Shubham Alock Singh** from the *Remedies In-Depth Course* (June 2023).

# 🚀 Deploy to GitHub Pages

## Option 1 — GitHub Website (Recommended)

1. Create a new GitHub repository named:

```text
temple-remedy
```

2. Upload `index.html` to the repository root.

3. Open:

```text
Settings → Pages
```

4. Under **Source**, choose:

```text
Deploy from a branch
```

5. Select:

```text
Branch: main
Folder: /(root)
```

6. Click **Save**.

Your website will be live at:

```text
https://YOUR-USERNAME.github.io/temple-remedy/
```

---

## Option 2 — Git CLI

```bash
# Clone or create repository
git clone https://github.com/YOUR-USERNAME/temple-remedy.git
cd temple-remedy

# Copy project file
cp /path/to/index.html .

# Commit and push
git add index.html
git commit -m "Add Temple Remedy web app"
git push origin main
```

Then enable GitHub Pages from:

```text
Settings → Pages → main branch
```

# 🔧 Important SEO Setup

After deployment, replace all instances of:

```text
YOUR-USERNAME
```

with your actual GitHub username inside `index.html`.

Example:

```bash
sed -i 's/YOUR-USERNAME/yourusername/g' index.html
```

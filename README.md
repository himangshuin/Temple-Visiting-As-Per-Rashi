# 🕉 Temple Visiting as Remedy — Vedic Astrology Temple Recommendation Web App

A spiritually inspired and SEO-optimized Hindu temple recommendation web application that helps devotees discover the most suitable temples to visit according to their **Vedic Birth Sign (Rashi)**.

This project combines the wisdom of **Vedic Astrology**, **Temple Remedies**, **Sanatan Dharma traditions**, and sacred pilgrimage systems into a modern, lightweight, and fully responsive web experience.

The application recommends sacred temples from multiple Hindu traditions including:

* Shiva Jyotirlingas
* Devi Shaktipeeths
* Ganesha Temples
* Vishnu Temples
* Rama Temples
* Hanuman Temples
* Krishna & Radha Temples

Built using pure HTML, CSS, and JavaScript with zero dependencies, this project is designed for spiritual seekers, astrology enthusiasts, devotees, pilgrims, and developers interested in Vedic astrology and Hindu temple systems.

# 🚀 GitHub Pages Deployment Guide

## Option 1 — Deploy Using GitHub Website

### Step 1

Create a new GitHub repository:

```text
Temple-Visiting-As-Per-Rashi
```

---

### Step 2

Upload all project files to the repository root.

---

### Step 3

Open:

```text
Settings → Pages
```

---

### Step 4

Under **Build and deployment**, choose:

```text
Deploy from a branch
```

---

### Step 5

Select:

```text
Branch: main
Folder: /(root)
```

---

### Step 6

Click:

```text
Save
```

Your website will become live at:

```text
https://YOUR-USERNAME.github.io/Temple-Visiting-As-Per-Rashi/
```
